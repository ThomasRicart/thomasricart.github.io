            Le réseau de gauche et le réseau de droite ne peuvent pas encore communiquer car les paquets ne savent pas par où sortir du sous réseau.

Par exemple, pour le sous-réseau de gauche (\verb+192.168.1.0/24+), la passerelle par défaut sera l'adresse IP du port du routeur connecté à ce sous-réseau. De même, pour le sous-réseau de droite (\verb+172.16.0.0/16+), la passerelle par défaut sera l'adresse IP du port du routeur connecté à ce sous-réseau.

    \begin{exercice_cours}[Mettre en place un serveur ftp]{}

        \begin{definition}{Le serveur ftp}{}
            Le serveur \verb+ftp+ (File Transfert Protocol) permet de transférer des fichiers par Internet. Toute personne ayant l'autorisation peut télécharger ou envoyer des fichiers depuis un ordinateur distant.
        \end{definition}

        \begin{enumerate}
            \item Dans le serveur HTTP, cliquer sur ``Config / FTP''. Un utilisateur est déjà renseigné. 
            
            Noter l'identifiant et le mot de passe: \dotfill
            \item Sur un des PC, ouvrir l'invite de commande et connectez vous au serveur \verb+ftp+ en tapant \verb+ftp 172.16.0.15+. Entrer l'identifiant et le mot de passe.
            \item La commande \verb+dir+ permet de lister les fichiers sur le serveur \verb+ftp+
            \item Pour ajouter un fichier vous pouvez utiliser la commande \verb+put+. A la racine de votre machine virtuelle vous avez un fichier nommé ``\verb+sampleFile.txt+''. Transférez le sur le serveur \verb+ftp+ avec ``\verb+put sampleFile.txt+''.
            \item Vérifier que le transfert s'est bien passé en listant les fichiers présents avec \verb+dir+.
            \item Récupérez un autre fichier depuis le serveur \verb+ftp+ avec ``\verb+get <nom du fichier>+''.
            \item Quitter le serveur \verb+ftp+ en faisant \verb-Control + C-.
            \item Vérifier que vous avez bien récupéré le fichier sur le disque local de la machine avec la commande \verb+dir+.
        \end{enumerate}
    \end{exercice_cours}
