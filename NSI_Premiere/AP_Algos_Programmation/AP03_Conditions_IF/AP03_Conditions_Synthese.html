AP03 : Les Structures Conditionnelles

NSI Première - Maîtriser les tests et les branchements logiques

1. La structure de base : IF

Le bloc if permet d'exécuter un code uniquement si une condition est Vraie (True). L'indentation est obligatoire.

if total > 0:
    total = total + 1 # Exécuté si la condition est vraie
print(total) # Exécuté dans tous les cas

2. Alternative : IF / ELSE

On utilise else pour définir ce qui doit se passer quand la condition est Fausse (False).

def valeur_absolue(x):
    if x <  0:
        return -x
    else:
        return x

3. Conditions multiples : IF / ELIF / ELSE

Quand il y a plus de deux possibilités, elif (contraction de "else if") permet d'enchaîner les tests.

if x > 0:
    sortie = 1
elif x == 0:
    sortie = 0
else:
    sortie = -1

4. Les opérateurs de comparaison

Essentiels pour construire les conditions :

  • == : Égal à (attention, ne pas confondre avec = d'affectation)
  • != : Différent de
  • < , > : Inférieur / Supérieur strict
  • <= , >= : Inférieur / Supérieur ou égal
💡 À retenir : En Python, dès qu'une condition d'un bloc if ou elif est validée, les blocs suivants du même groupe (les autres elif ou le else) sont ignorés.
