\begin{definition}[Bases de données]{}
	
\begin{itemize}
	\item Une \textbf{base de données} (BD) peut contenir une ou plusieurs \textbf{tables}.
	\item Une \textbf{table} est un ensemble d'objets appelés \textbf{enregistrements}.
	\item Un \textbf{enregistrement} possède différentes \textbf{valeurs} associées à des \textbf{attributs} (descripteurs).
	\item Toutes les \textbf{valeurs} que peut prendre un attribut s'appelle son \textbf{domaine de valeurs} et sont toutes du même type.
\end{itemize}
\begin{center}
\includegraphics[scale=0.8]{exemple_table.png}
\end{center}
\end{definition}



\begin{easybox}[Le format CSV]{}
Le format CSV (pour comma separated values, valeurs séparées par des virgules) est un fichier texte où chaque ligne représente un enregistrement avec différents champs séparés par une virgule. La première ligne d'un fichier CSV est généralement utilisée pour indiquer le nom des différents champs que l'on appelle les descripteurs.
\end{easybox}
\begin{itemize}
	\item     Le caractère de séparation peut aussi être un point-virgule, une tabulation ou deux points.
	\item     Un fichier CSV peut être lu par un éditeur de texte mais aussi un tableur.

\end{itemize}

 Dans la suite du cours, nous utiliserons comme BD, le fichier \verb|countries.csv| issues du site \verb|www.geonames.org| qui listes les pays du monde entier. \section{Importation d'une table depuis un fichier CSV}

Le module natif csv de Python permet d'importer les données depuis un fichier CSV.
Voici un script permettant de charger le fichier \verb|countries.csv| avec des points-virgules comme délimitations.


\begin{CodePythontexAlt}[Largeur=1\linewidth,Centre,PremLigne=1, lineNumbers=false]{}
import csv

with open('countries.csv', newline='', encoding='utf-8') as csvfile:
    pays = list(csv.reader(csvfile, delimiter=';'))
\end{CodePythontexAlt}

\section{Trier une table}

Le tri d'une table présente deux intérêts: il permet d'établir des classements et accélère la recherche de l'information.
En effet, il est beaucoup plus efficace (rapide) d'effectuer une recherche dans une liste triée. Cela vient du fait qu'il est alors possible d'effectuer une recherche par dichotomie. 

\subsubsection{Tri selon un unique critère}

 On ne peut pas directement trier la liste pays car cela ne veut rien dire. Il faut indiquer selon quels critères (population, superficie etc.) on veut effectuer ce tri.

Pour cela, on appelle la fonction \verb|sorted(list)| qui retourne une copie de la liste triée ou la méthode \verb|list.sort()| qui trie la liste sur place.
Ces deux fonctions ont deux paramètres possibles :

\begin{itemize}
	\item     \verb|key| qui est la fonction permettant d'évaluer les éléments de la liste afin de les classer selon leur score
	\item     \verb|reverse| qui est un boolean (par défaut \verb|False|) permettant de trie selon l'ordre décroissant (\verb|True|).

\end{itemize}

Par exemple, si l'on veut trier les pays selon leur superficie: il faut d'abord définir une fonction qui permet d'accéder à la valeur de l'attribut \verb|'area'| d'un enregistrement puis retourne le score de l'enregistrement afin qu'il puisse être comparé aux autres et ainsi trier la liste entière.


\begin{CodePythontexAlt}[Largeur=1\linewidth,Centre,PremLigne=1, lineNumbers=false]{}
def cle_superficie(p):
    return p['area']

pays.sort(key=cle_superficie, reverse=True) # tri

top5 = []
for p in pays[:5] : # extraction
	top5.append((p['country'], p['area']))
\end{CodePythontexAlt}

\subsubsection{Tri selon plusieurs critères}

 Supposons maintenant que l'on veut trier les pays selon deux critères: d'abord par continent, puis par population.

\begin{CodePythontexAlt}[Largeur=1\linewidth,Centre,PremLigne=1, lineNumbers=false]{}
def cle_population(p):
    return int(p['population'].replace(",", ""))

def cle_continent(p):
    return p['continent']

pays.sort(key=cle_population, reverse=True)  # tri par population
pays.sort(key=cle_continent)                 # tri par continent
resultat = []
for p in pays[:5] : # extraction
	resultat.append((p['country'], p['population'], p['continent']))
for p in resultat : # affichage
    print(p)
\end{CodePythontexAlt}

\section{Ecrire dans un fichier CSV}

Pour écrire dans un fichier CSV, on peut utiliser la fonction \verb|csv.writer| du module csv. Voici un script permettant d'écrire une liste de listes dans un fichier CSV avec des points-virgules comme délimiteurs.	
\begin{CodePythontexAlt}[Largeur=1\linewidth,Centre,PremLigne=1, lineNumbers=false]{}
import csv	
with open('nom_fichier.csv', 'w', newline='', encoding='utf-8') as csvfile:
	writer = csv.DictWriter(csvfile, fieldnames=liste_de_dico[0].keys(), delimiter=';')
	writer.writeheader()
	writer.writerows(liste_de_dico)
\end{CodePythontexAlt}
