LP05 : Les Listes et les Tuples

NSI Première - Structures de données séquentielles

1. Les Listes (Type list)

Une liste est une structure mutable (modifiable) d'éléments ordonnés, entourés de crochets [].

ma_liste = [10, 20, 30] # Création
print(ma_liste[0]) # Accès par indice (affiche 10)
ma_liste[1] = 99 # Modification autorisée (Mutable)
print(len(ma_liste)) # Nombre d'éléments (affiche 3)
Attention : Les indices commencent toujours à 0. L'indice -1 permet d'accéder au dernier élément.

2. Méthodes et Itération

On peut parcourir une liste avec une boucle for ou la modifier avec des méthodes spécifiques.

# Ajouter et supprimer
L = [1, 2]
L.append(3) # L devient [1, 2, 3]
dernier = L.pop() # Supprime et renvoie 3

# Parcourir la liste
for elt in L:
    print(elt)

3. Les Tuples (Type tuple)

Un tuple est une structure immuable (non modifiable) entourée de parenthèses ().

mon_tuple = (1, 2, 3)
print(mon_tuple[0]) # Accès possible
# mon_tuple[0] = 10 -> ERREUR (Immuable)

4. Effets de bord et Slicing

Affecter une liste à une nouvelle variable crée une référence, pas une copie.

L1 = [1, 2]
L2 = L1 # Simple référence
L3 = L1[:] # Vraie copie par "Slicing"
