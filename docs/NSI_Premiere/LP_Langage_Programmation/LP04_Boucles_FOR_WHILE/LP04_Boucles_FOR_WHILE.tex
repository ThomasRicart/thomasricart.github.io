			\item Écrire une fonction \verb|puissance(x, n)| qui renvoie $x^n$. Par exemple, \verb|puissance(2, 5)| doit renvoyer $2^5=32$.	
			\item Écrire une fonction \verb|somme_puissances(n)| qui renvoie la somme des puissances de 2 de 0 à $n$ inclus, c'est à dire $2^0+2^1+2^2+\dots+2^n$. Par exemple, \verb|somme_puissances(5)| doit renvoyer $1+2+4+8+16+32=63$.

			\begin{itemize}
				\item 			Écrire un programme en Python pour savoir au bout de combien d'années il pourra avoir son scooter
				\item 			Modifier le programme pour que l'utilisateur puisse choisir le nombre d'années et la somme initiale.
				\item 			Modifier le programme pour que l'utilisateur puisse également choisir le taux d'intéret en pourcentage.

			\end{itemize}
