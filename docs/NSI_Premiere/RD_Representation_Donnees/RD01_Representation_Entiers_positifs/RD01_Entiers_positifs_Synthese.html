RD01 : Représentation des Entiers Positifs

Architecture des ordinateurs & Codage de l'information

I. Origine Physique : Le Binaire

L'informatique repose sur des composants électroniques à deux états exclusifs. On utilise le système binaire (Base 2) dont l'unité est le bit (binary digit).

  • États : Passant (1) ou Bloqué (0) pour un transistor.
  • Capacité de codage : Sur $n$ bits, on peut représenter $2^n$ valeurs distinctes.
  • Plage de valeurs : Pour $n$ bits, l'intervalle est $[0 ; 2^n - 1]$.
Exemple : Un octet (8 bits) permet de coder $2^8 = 256$ valeurs, allant de $0$ à $255$.

II. Systèmes de Numération (Base $b$)

Tout entier naturel $n$ peut se décomposer selon une base $b$ sous la forme d'une somme de puissances :

$n = \sum_{k=0}^{N} a_k b^k = a_N b^N + \dots + a_1 b^1 + a_0 b^0$

1. Le système Binaire (Base 2)

Utilise les chiffres $\{0, 1\}$.

# Conversion 1101 (Binaire) vers Décimal :
1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8 + 4 + 0 + 1 = 13

2. Le système Hexadécimal (Base 16)

Utilise $\{0, \dots, 9, A, B, C, D, E, F\}$. Un symbole hexadécimal code exactement 4 bits (un quartet).

Décimal101112131415
HexaABCDEF
Binaire101010111100110111101111

III. Méthodes de Conversion

Décimal → Binaire/Hexa : Effectuer des divisions euclidiennes successives par la base (2 ou 16) et lire les restes de bas en haut.
Binaire ↔ Hexadécimal : Regrouper les bits par 4 en partant de la droite.
Exemple : 1101 1010 → 1101 (D) et 1010 (A) → DA16.

IV. Manipulation en Python

Python propose des fonctions natives pour convertir ces bases :

a = 44 # Décimal
bin_a = bin(a) # '0b101100'
hex_a = hex(a) # '0x2c'
dec_a = int("1011", 2) # 11
