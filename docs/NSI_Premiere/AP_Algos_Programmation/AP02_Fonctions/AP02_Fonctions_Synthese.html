AP02 : Les Fonctions

NSI Première - Modularité et réutilisation du code

1. Gestion des Entrées (Saisie utilisateur)

La fonction input() permet d'interrompre le programme pour attendre une saisie au clavier. Attention : la valeur récupérée est toujours de type str.

nom = input("Quel est ton nom ?") # Type str
age = int(input("Ton âge ?")) # Conversion immédiate en entier

2. Définir et Appeler une Fonction

Une fonction est un bloc de code réutilisable. On la définit avec le mot-clé def et elle renvoie généralement un résultat avec return.

def calculer_carre(x):
    """Renvoie le carré d'un nombre x."""
    resultat = x ** 2
    return resultat

# Appel de la fonction
val = calculer_carre(4) # val stocke 16
Important : Ne pas confondre print() (qui affiche dans la console) et return (qui renvoie une valeur utilisable par le reste du programme).

3. Portée des variables : Locales vs Globales

La visibilité d'une variable dépend de l'endroit où elle est déclarée :

  • Variables locales : Définies à l'intérieur d'une fonction, elles n'existent pas à l'extérieur.
  • Variables globales : Définies dans le corps principal du programme, elles sont visibles partout.
g = 10 # Variable globale

def ma_fonction():
    L = 5 # Variable locale
    print(g) # Fonctionne car g variable globale

ma_fonction()
print(L) # ERREUR : L n'est pas connue ici

4. Validation avec Assertions

L'instruction assert permet de vérifier que le code produit le résultat attendu. C'est une étape essentielle pour tester ses fonctions.

def somme(a, b):
    return a + b

assert somme(6, 8) == 14 # Si vrai, rien ne se passe
assert somme(1, 2) == 5 # Déclenche une AssertionError

5. Affichages complexes

On peut combiner texte et variables dans un print() en utilisant la virgule comme séparateur.

a, b = 6, 4
print("La somme de", a, "et", b, "est", a + b)
