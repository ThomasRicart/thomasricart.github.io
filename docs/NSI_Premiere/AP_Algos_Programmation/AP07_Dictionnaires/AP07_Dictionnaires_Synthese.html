AP07 : Les Dictionnaires

NSI Première - Structures de données Clé-Valeur

1. Définition et Création

Un dictionnaire est une collection non ordonnée et mutable stockant des paires clé: valeur.

# Création
mon_dico = {"nom": "Doe", "age": 17}
# Accès
print(mon_dico["nom"]) # Affiche Doe
Les clés doivent être uniques et de type immuable (chaînes, nombres, tuples).

2. Méthodes et Opérations

  • del dico[cle] : Supprime une paire.
  • dico.keys() : Renvoie la liste des clés.
  • dico.values() : Renvoie la liste des valeurs.
  • dico.items() : Renvoie les couples (clé, valeur).

3. Dictionnaire par compréhension

carres = {x: x**2 for x in range(5)}
# Résultat : {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}
