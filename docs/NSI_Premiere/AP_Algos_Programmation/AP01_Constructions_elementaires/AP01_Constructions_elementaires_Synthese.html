LP01 : Constructions Élémentaires

NSI Première - Les bases du langage Python

1. Variables et Types de données

Une variable permet de stocker une information en mémoire. En Python, le type est déterminé automatiquement lors de l'affectation.

Type Nom Python Exemple
Entier int age = 17
Flottant (décimal) float prix = 19.99
Chaîne de caractères str nom = "Alice"
Booléen bool est_pret = True
Règle de nommage : Un nom de variable doit commencer par une lettre ou _. Il ne peut pas commencer par un chiffre ni contenir d'espaces ou de caractères spéciaux (sauf _).

2. Opérations Numériques

a = 7
b = 3

div_classique = a / b # 2.333... (float)
div_entiere = a // b # 2 (quotient entier)
reste = a % b # 1 (modulo)
puissance = a ** b # 7³ = 343

3. Manipulation des Chaînes (Slicing)

On accède aux caractères par leur indice (le premier est 0).

mot = "Numérique"
caractere = mot[0] # "N"
sous_chaine = mot[1:4] # "umé" (indices 1, 2, 3)
inverse = mot[::-1] # "euqirémuN"
longueur = len(mot) # 9

4. Logique et Comparaisons

Les opérateurs logiques permettent de combiner des conditions :

  • and : Vrai si les deux sont vrais.
  • or : Vrai si au moins l'un est vrai.
  • not : Inverse la valeur (le contraire).
x = 10
test = (x > 5) and (x <= 15) # True
different = (x != 10) # False

5. Transtypage (Casting)

Il est parfois nécessaire de forcer le changement de type, notamment lors d'une saisie utilisateur :

saisie = "15"
nombre = int(saisie) + 5 # 20
texte = str(2024) # "2024"
