AP04 : Les Boucles FOR et WHILE

NSI Première - Répétitions bornées et non-bornées

1. La boucle bornée : FOR

On utilise for quand on connaît à l'avance le nombre de répétitions. La fonction range(a, b) génère les entiers de a jusqu'à b-1.

for i in range(2, 7):
    print(i) # Affiche 2, 3, 4, 5, 6 (5 tours)

for lettre in "NSI":
    print(lettre) # Itère sur chaque caractère

2. La boucle non-bornée : WHILE

La boucle while s'exécute tant que la condition est vraie. On ne sait pas forcément à l'avance quand elle s'arrêtera.

n = 0
while n < 10000:
    n = n * 2 + 1
    print(n)
